## ✨ Features

- 💼 **Portfolio Overview**: View ETH balance, wallet identity (ENS/address), and holdings at a glance.
- 🪙 **Token Management**: Dynamic token list rendering from wallet portfolio data.
- ⚡ **Quick Actions Hub**: Send, Receive, Buy, and Swap flows in an intuitive UI.
- 🧭 **Smooth Navigation**: Built with React Navigation for scalable architecture.
- 🎨 **Clean UI**: Focused on usability, performance, and modern look and feel.

## 🛠 Tech Stack

- React Native `0.83`
- TypeScript
- React Navigation
- Ethers.js

## 🚀 Getting Started

### 1) 📦 Install Dependencies

```sh
npm install
```

### 2) 🛰 Start Metro Bundler

```sh
npm start
```

### 3) ▶️ Run the App

```sh
# Android
npm run android

# iOS
npm run ios
```

### 🍎 iOS Setup (First Time Only)

```sh
cd ios && pod install && cd ..
```

## 📂 Project Structure

```text
src/
├── pages/       # App screens
├── Contexts/    # Global state and wallet context
└── constants/   # Design system and configs
```

## 🎯 About This Project

This project provides a production-ready wallet UI foundation with scalable structure and real-world Web3 use cases in mind. It can be extended into a full DeFi wallet with on-chain integrations.
