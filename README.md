# 🚀 WalletApp

WalletApp is a modern React Native Web3 wallet UI focused on a clean portfolio experience, fast wallet actions, and seamless token management.

## 📸 Preview

Add app screenshots or GIFs here for a stronger project presentation.

## ✨ Features

- 💼 **Portfolio Overview**: View ETH balance, wallet identity (ENS/address), and holdings at a glance.
- 🪙 **Token Management**: Dynamic token list rendering from wallet portfolio data.
- ⚡ **Quick Actions Hub**: Send, Receive, Buy, and Swap flows in an intuitive UI.
- 🧭 **Smooth Navigation**: Built with React Navigation for scalable architecture.
- 🎨 **Clean UI**: Focused on usability, performance, and modern look and feel.

## 🛠 Tech Stack

- React Native `0.83`
- TypeScript
- React Navigation
- Ethers.js

## 🚀 Getting Started

### 1) 📦 Install Dependencies

```sh
npm install
```

### 2) 🛰 Start Metro Bundler

```sh
npm start
```

### 3) ▶️ Run the App

```sh
# Android
npm run android

# iOS
npm run ios
```

### 🍎 iOS Setup (First Time Only)

```sh
cd ios && pod install && cd ..
```
